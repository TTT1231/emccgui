# Changelog

## 1.0.0

- Initial release of Emscripten Compiler GUI
- Visual configuration interface for Emscripten compiler options
- Real-time emcc command generation with one-click copy
- Code editor with syntax highlighting (CodeMirror 6)
- Built-in reference documentation for emcc options
- Conflict detection for mutually exclusive options
- Bilingual interface support (English/Chinese)
- Language switcher with state preservation
- Search functionality with highlighting
- Category-based option browsing
- Single HTML file distribution (works offline)
