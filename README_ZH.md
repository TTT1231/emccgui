# Emscripten 编译器 GUI

一个用于 Emscripten (emcc) 编译器的图形化配置界面，帮助开发者通过可视化方式构建编译命令，无需记忆复杂的命令行参数。

## 简介

Emscripten 能够将 C/C++ 代码编译为 WebAssembly，在浏览器中运行。但 emcc 编译器拥有数百个命令行选项，配置复杂且难以记忆。

本工具提供了一个直观的图形界面，让你可以：

- **可视化配置**: 通过分类浏览和选择编译选项，每个选项都有详细说明
- **冲突检测**: 自动检测选项之间的冲突关系，避免无效配置
- **命令生成**: 自动生成完整的 emcc 编译命令，一键复制到剪贴板
- **参考文档**: 内置编译器选项搜索功能，快速查找需要的参数

## 主要功能

| 功能 | 说明 |
|------|------|
| 文件选择 | 选择要编译的 C/C++ 源文件 |
| 选项配置 | 按类别浏览和配置编译选项（优化、输出、调试等） |
| 文档搜索 | 搜索和查阅 emcc 编译器选项文档 |
| 主题切换 | 支持深色/浅色主题 |

## 技术实现

基于 Vue 3 + TypeScript + Vite 构建，使用 CodeMirror 提供代码编辑体验。
